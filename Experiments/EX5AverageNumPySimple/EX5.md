# Experiment 5
## Aim
Demonstrate that different but equivalent implementations of a simple numerical routine does not affect reproducibility 
but may affect other measures.

## Hypothesis
We present four hypothesis
  - RR(G1, G2)
  - !RT(G1, G2)
  - RP(G1, G2)
  - RPLS(G1, G2)
  - !RPLC(G1, G2)
  
## Results
| Mode | Match |
|:------:|:----:|
| RR   | True |
| RT   | False |
| RP   | True |
| RPLS | True |
| RPLC | False |
